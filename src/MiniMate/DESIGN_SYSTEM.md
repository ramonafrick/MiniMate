# MiniMate Design System

## Color Scheme - Friendly & Child-Appropriate

### Primary Colors (Module-Specific)
```css
--color-weather: #4A90E2;      /* Blue - Weather, Information, Trust */
--color-calendar: #9013FE;     /* Purple - Calendar, Time, Creativity */
--color-profile: #7ED321;      /* Green - Profile, Success, Safe */
--color-clothing: #F5A623;     /* Orange - Clothing, Energy, Warning */
--color-home: #FF6B9D;         /* Pink - Home, Friendly, Welcoming */
```

### Secondary Colors (UI Elements)
```css
--color-primary: #4A90E2;      /* Primary interaction color */
--color-secondary: #7ED321;    /* Secondary actions */
--color-accent: #F5A623;       /* Accents, Highlights */
--color-success: #7ED321;      /* Success, Confirmation */
--color-warning: #F5A623;      /* Warning, Caution */
--color-error: #FF5252;        /* Error, Danger */
--color-info: #4A90E2;         /* Information */
```

## Design Principles

1. **Child-Friendly:** Bright, friendly colors with high saturation
2. **Consistent:** Each module has a unique color identity
3. **Accessible:** High contrast for good readability
4. **Playful:** Animations and gradients for added fun
5. **Professional:** Not too chaotic, clear structure
